Monster of the Week Python terminal game

Language focus: Python
Time to complete: 1 week

My first coding assignment, this project was written in Python and originally played in the terminal of VS Code. The learning objective of this project was simple programming fundementals, object-oriented programming, and learning a new language quickly.
The game itself is a text-based roleplaying game based on the tabletop roleplaying game Monster of the Week. This system is intended for use in narratives like Supernatural, Buffy the Vampire Slayer, and The X-Files.
For more information on the original tabletop game and my logic in building the project, check out my Medium article on the project.
What are you waiting for? Get out there and kill some monsters. Good hunting.

Barovian Blackjack

Language focus: JavaScript
Time to complete: 1 week

You have to start somewhere learning the web languages (HTML, CSS, and JavaScript) and webpage interactivity. So why not make it fun?
The exact goal of this assignment and the use case of the final project (although entertaining) is less important than the techniques and methods I learned while working on it -- mostly JavaScript and HTML/CSS fundementals and Document Object Model manipulation. I also learned how to work with an existing code base, as we were given a template of code to start with which we pulled from GitHub and weren't allowed to change. Lastly, I worked within tighter confines than my first project -- this one had to be a blackjack game with all the functionality of standard blackjack.
In this project, you play blackjack with standard casino rules and a non-standard card deck. I used some assets leftover from my D&D days in this one, particularly the Tarokka deck which appears in the Curse of Strahd adventure book and an animated background made for play in the same adventure from Beneos Battlemaps.
Test your luck!

Front-end Poetry (Alche-traz)

Language focus: CSS (via TailwindCSS), asynchronous JavaScript
Time to complete: 1.5 weeks

It was with this project that I began to start truly using the internet in my webpages -- I learned how to use APIs/databases and their contents on live webpages. The final product was a web page that allows the user to search and generate a bunch of different poetry.
This project was powered by Thundercomb's wonderful Poetry Database. I wrote another my Medium article on this project to explain the mechanisms and my thinking behind it.
