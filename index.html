Monster of the Week Python terminal game

Language focus: Python Time to complete: 1 week

My first coding assignment, this project was written in Python and originally played in the terminal of VS Code. The learning objective of this project was simple programming fundementals, object-oriented programming, and learning a new language quickly.
The game itself is a text-based roleplaying game based on the tabletop roleplaying game Monster of the Week.
For more information on the original tabletop game and my logic in building the project, check out my article on the project. Good hunting.

Madam Eva's Blackjack

Language focus: JavaScript Time to complete: 1 week

You have to start somewhere learning webpage interactivity. The exact goal of this assignment and the use case of the final project (although fun) is less important than the techniques and methods I learned while working on it -- mostly JavaScript and HTML fundementals and Document Object Model manipulation.
In this project, you play blackjack with standard casino rules and a non-standard card deck. I used some assets leftover from my D&D days in this one, particularly the Tarokka deck which appears in the Curse of Strahd adventure book and an animated background made for play in the same adventure from Beneos Battlemaps.
You can test your luck here.
